# Forensic Biometrics Studio

Forensic Biometrics Studio is an open-source tool designed for the side-by-side comparison of forensic evidence images (e.g., fingerprints, shoeprints, or earprints).

The software allows users to open two images side by side, mark corresponding features on both images, compare the marked traces, and generate a structured comparison report. It is designed especially for forensic domains where dedicated comparison software is not widely available, including but not limited to forensic otoscopy, footwear mark analysis, dactyloscopy.

### The project has two main goals:

* to **support forensic experts** in the documentation and **comparison** of trace evidence where dedicated software is often lacking (e.g., forensic otoscopy).
* to **support education, training, and research** in forensic image comparison. Serving as a transparent, accessible platform for students and researchers to learn the principles of biometric identification.

## 📜 Usage Terms & Citation

Forensic Biometrics Studio is developed as an open academic project.

To help us document the impact of the project, maintain the software, and report its use in institutional and grant-related summaries, organisations using the software for commercial, institutional, or educational purposes are expected to notify the project maintainers once every four years.

### Notification of Use

Entities utilizing Forensic Biometrics Studio for **commercial, scientific or educational purposes** are encouraged to notify the project maintainers of their usage once every four years. This information is vital for our reporting and helps us secure continued support for the project.

Please send a brief notification to: [biometrics@ubb.edu.pl](mailto:biometrics@ubb.edu.pl)

You can use one of the templates listed below:
